💬 评论系统配置说明

本站使用 Giscus 评论系统（基于 GitHub Discussions）。
在 管理后台 → 设置 中填入你的 Giscus 配置信息即可启用。

配置步骤：
1. 前往 giscus.app 生成配置代码
2. 复制 data-repo、data-repo-id、data-category、data-category-id
3. 粘贴到管理后台 → 设置 → 评论系统配置
